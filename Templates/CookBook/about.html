About CookBook

Recipe Management & Grocery List Generator

Project Overview

CookBook is a comprehensive recipe management system that simplifies meal planning and grocery shopping. Browse recipes, select your favorites, and automatically generate consolidated grocery lists without duplicates.

Key Features

  • Browse recipes with search and filters
  • Select multiple recipes for meal planning
  • Automatic grocery list generation
  • Duplicate ingredient elimination
  • Interactive checkbox shopping list
  • Dark mode support
  • RESTful API endpoints
  • Responsive design
Backend
  • Python 3.9+
  • Django 4.2
  • Django REST Framework
  • PostgreSQL
Frontend
